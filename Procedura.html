Programmazione ad Oggetti

Procedura d'esame

Si possono usare versioni cartacee delle slide ed altro materiale.
NON si possono usare drive USB, CD-ROM etc.
NON è possibile uscire prima della fine del compito, anche se ci si ritira.

Leggere e seguire attentamente le istruzioni.

Laboratorio

In laboratorio viene fornito un file zip contenente un progetto Eclipse ed il testo dell'esame.
  • Installare Eclipse e Java
  • Aprire Eclipse fornendo la matricola e un indirizzo di email valido (ad esempio Smatricola@studenti.polito.it)
  • Importare il progetto seguendo le istruzioni. (vedi: http:\\softeng.polito.it\02CBI\ProceduraEsame.html)
    • Installare il programma Power Archive
    • Scaricare il file zip dall'indirizzo sottostante
    • Estrarre il file zip in un folder a piacere (non il desktop)
    • Da Eclipse: File; Import; Existing project into workspace
  • Svolgere il compito senza modificare nomi, tipi e visibilità di classi e metodi forniti.
  • Prima della fine esportare il progetto (click destro sul progetto, export, as zip, P:\matricola-cognome-nome.zip)
  • Alla fine dell'esame sottomettere (click destro sul progetto, Sottometti progetto)
    • NB il progetto sottomesso DEVE essere sintatticamente corretto, ovvero deve compilare.

Casa

Per avere il compito valutato occorre sottomettere la correzione.
  • Prelevare i test che saranno pubblicati sul sito del corso dopo l'esame
  • Recuperare il progetto sottomesso in laboratorio (che viene inviato automaticamente via email)
  • Apportare le modifiche (minime) necessarie a far passare i test
  • Sottomettere il progetto corretto all'indirizzo web indicato sul sito del corso entro la data indicata sul sito
